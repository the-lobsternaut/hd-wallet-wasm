API Reference

Complete JavaScript/TypeScript API documentation for HD Wallet WASM.

Installation

bash 
# NPM
npm install hd-wallet-wasm

# Yarn
yarn add hd-wallet-wasm

# PNPM
pnpm add hd-wallet-wasm

Initialization

The WASM module must be initialized before use. This is an async operation that loads and compiles the WebAssembly binary.

javascript 
import HDWalletWasm from 'hd-wallet-wasm';

// Initialize the module
const wallet = await HDWalletWasm();

// Check module status
console.log('Version:', wallet.getVersion());
console.log('Has Crypto++:', wallet.hasCryptopp());
console.log('FIPS Mode:', wallet.isFipsMode());
console.log('Supported Coins:', wallet.getSupportedCoins());
console.log('Supported Curves:', wallet.getSupportedCurves());

// In WASI environments, inject entropy first
if (wallet.getEntropyStatus() === 0) {
  const entropy = crypto.getRandomValues(new Uint8Array(64));
  wallet.injectEntropy(entropy);
}

Enums & Types

Curve

Supported elliptic curves for key derivation and signing.

enum Curve {
  SECP256K1 = 0,  // Bitcoin, Ethereum
  ED25519 = 1,    // Solana, Polkadot
  P256 = 2,       // NIST P-256 (secp256r1)
  P384 = 3,       // NIST P-384 (secp384r1)
  X25519 = 4      // Key exchange only
}

Bitcoin address format types.

enum BitcoinAddressType {
  P2PKH = 0,   // Legacy (1...)

Mnemonic API

mnemonic.generate() sync

Generate a random BIP-39 mnemonic phrase with the specified word count.

Parameters
wordCount number Number of words (12, 15, 18, 21, or 24). Default: 24
language Language Wordlist language. Default: ENGLISH
Returns
mnemonic string Space-separated mnemonic phrase
javascript 
// Generate 24-word English mnemonic
const mnemonic24 = wallet.mnemonic.generate(24);

// Generate 12-word mnemonic
const mnemonic12 = wallet.mnemonic.generate(12);

// Generate Japanese mnemonic
const mnemonicJp = wallet.mnemonic.generate(24, Language.JAPANESE);
mnemonic.validate() sync

Validate a BIP-39 mnemonic phrase. Checks word count, wordlist membership, and checksum.

Parameters
mnemonic string Space-separated mnemonic phrase to validate
language Language Wordlist language. Default: ENGLISH
Returns
isValid boolean True if mnemonic is valid
javascript 
const isValid = wallet.mnemonic.validate(mnemonic);
if (!isValid) {
  throw new Error('Invalid mnemonic phrase');
}
mnemonic.toSeed() sync

Convert a mnemonic phrase to a 64-byte seed using PBKDF2-HMAC-SHA512.

Parameters
mnemonic string Space-separated mnemonic phrase
passphrase string Optional passphrase (25th word). Default: ""
Returns
seed Uint8Array 64-byte seed
javascript 
// Without passphrase
const seed = wallet.mnemonic.toSeed(mnemonic);

// With passphrase (25th word)
const seedWithPass = wallet.mnemonic.toSeed(mnemonic, 'my secret passphrase');

console.log('Seed length:', seed.length); // 64
mnemonic.toEntropy() sync

Convert a mnemonic phrase back to its original entropy bytes.

javascript 
const entropy = wallet.mnemonic.toEntropy(mnemonic);
console.log('Entropy bytes:', entropy.length); // 16-32 depending on word count
mnemonic.fromEntropy() sync

Generate a mnemonic phrase from provided entropy bytes.

javascript 
// Create entropy externally (CSPRNG)
const entropy = crypto.getRandomValues(new Uint8Array(32)); // 256 bits

// Convert to 24-word mnemonic
const mnemonic = wallet.mnemonic.fromEntropy(entropy);
console.log(mnemonic.split(' ').length); // 24

HD Key API

hdkey.fromSeed() sync

Create a master HD key from a seed. This is the root of the key derivation tree.

Parameters
seed Uint8Array 64-byte seed from mnemonic.toSeed()
curve Curve Elliptic curve. Default: SECP256K1
javascript 
const seed = wallet.mnemonic.toSeed(mnemonic);

// Create secp256k1 master key (Bitcoin, Ethereum)
const masterKey = wallet.hdkey.fromSeed(seed);

// Create Ed25519 master key (Solana)
const masterKeyEd = wallet.hdkey.fromSeed(seed, Curve.ED25519);
hdkey.fromXprv() / fromXpub() sync

Parse an extended key from its Base58Check-encoded serialization.

javascript 
// Parse extended private key
const key = wallet.hdkey.fromXprv('xprv9s21ZrQH143K...');

// Parse extended public key (watch-only)
const pubKey = wallet.hdkey.fromXpub('xpub661MyMwAqRbc...');
console.log(pubKey.isNeutered); // true
key.derivePath() sync

Derive a child key at the specified BIP-32 path.

javascript 
// Derive Bitcoin (BIP-84 native SegWit)
const btcKey = masterKey.derivePath("m/84'/0'/0'/0/0");

// Derive Ethereum
const ethKey = masterKey.derivePath("m/44'/60'/0'/0/0");

// Access key data
console.log('Depth:', btcKey.depth);              // 5
console.log('Path:', btcKey.path);                // "m/84'/0'/0'/0/0"
console.log('Private key:', btcKey.privateKey()); // Uint8Array(32)
console.log('Public key:', btcKey.publicKey());   // Uint8Array(33)
key.deriveChild() / deriveHardened() sync

Derive a single child key at the specified index.

javascript 
// Non-hardened derivation (index 0-2147483647)
const child0 = masterKey.deriveChild(0);

// Hardened derivation (index >= 2147483648)
const childH = masterKey.deriveHardened(44);
// Equivalent to:
const childH2 = masterKey.deriveChild(44 + 0x80000000);
key.neutered() sync

Get a public-only (neutered) version of the key. Useful for creating watch-only wallets.

javascript 
const accountKey = masterKey.derivePath("m/84'/0'/0'");
const watchOnly = accountKey.neutered();

console.log(watchOnly.isNeutered);  // true
console.log(watchOnly.toXpub());    // "xpub..."
watchOnly.privateKey();             // throws Error
key.toXprv() / toXpub() sync

Serialize the extended key to Base58Check format.

javascript 
const xprv = masterKey.toXprv();  // "xprv9s21ZrQH143K..."
const xpub = masterKey.toXpub();  // "xpub661MyMwAqRbc..."

// Clean up sensitive data when done
masterKey.wipe();

Bitcoin API

bitcoin.getAddress() sync

Generate a Bitcoin address from a public key.

javascript 
const btcKey = masterKey.derivePath("m/84'/0'/0'/0/0");
const pubKey = btcKey.publicKey();

// Native SegWit (bech32)
const p2wpkh = wallet.bitcoin.getAddress(pubKey, BitcoinAddressType.P2WPKH);
// "bc1q..."

// Legacy
const p2pkh = wallet.bitcoin.getAddress(pubKey, BitcoinAddressType.P2PKH);
// "1..."

// Taproot
const p2tr = wallet.bitcoin.getAddress(pubKey, BitcoinAddressType.P2TR);
// "bc1p..."

// Testnet
const testnet = wallet.bitcoin.getAddress(pubKey, BitcoinAddressType.P2WPKH, Network.TESTNET);
// "tb1q..."
bitcoin.signMessage() sync

Sign a message using the Bitcoin Signed Message format.

javascript 
const message = 'Hello, Bitcoin!';
const signature = wallet.bitcoin.signMessage(message, btcKey.privateKey());

// Verify
const isValid = wallet.bitcoin.verifyMessage(message, signature, address);
console.log(isValid); // true

Ethereum API

ethereum.getAddress() sync

Generate an Ethereum address from a public key.

javascript 
const ethKey = masterKey.derivePath("m/44'/60'/0'/0/0");
const pubKey = ethKey.publicKey();

// Get address (lowercase)
const address = wallet.ethereum.getAddress(pubKey);
// "0x..."

// Get checksummed address (EIP-55)
const checksummed = wallet.ethereum.getChecksumAddress(address);
// "0x..." with mixed case
ethereum.signMessage() sync

Sign a message using EIP-191 personal_sign format.

javascript 
const message = 'Sign this message';
const signature = wallet.ethereum.signMessage(message, ethKey.privateKey());

// Recover signer address
const recoveredAddress = wallet.ethereum.verifyMessage(message, signature);
console.log(recoveredAddress === address); // true

Solana API

solana.getAddress() sync

Generate a Solana address from an Ed25519 public key.

javascript 
// Solana uses Ed25519 curve
const solSeed = wallet.mnemonic.toSeed(mnemonic);
const solMaster = wallet.hdkey.fromSeed(solSeed, Curve.ED25519);

// Standard Solana derivation path
const solKey = solMaster.derivePath("m/44'/501'/0'/0'");
const pubKey = solKey.publicKey();

// Get Base58 address
const address = wallet.solana.getAddress(pubKey);
// Base58-encoded public key

Cosmos API

cosmos.getAddress() sync

Generate a Cosmos/Tendermint address with custom bech32 prefix.

javascript 
const cosmosKey = masterKey.derivePath("m/44'/118'/0'/0/0");
const pubKey = cosmosKey.publicKey();

// Cosmos Hub
const cosmosAddr = wallet.cosmos.getAddress(pubKey, 'cosmos');
// "cosmos1..."

// Osmosis
const osmoAddr = wallet.cosmos.getAddress(pubKey, 'osmo');
// "osmo1..."

// Terra
const terraAddr = wallet.cosmos.getAddress(pubKey, 'terra');
// "terra1..."

Utilities

Hashing Functions

javascript 
const data = new TextEncoder().encode('Hello, World!');

// SHA-256
const sha256Hash = wallet.utils.sha256(data);

// SHA-512
const sha512Hash = wallet.utils.sha512(data);

// Keccak-256 (Ethereum)
const keccakHash = wallet.utils.keccak256(data);

// RIPEMD-160
const ripemdHash = wallet.utils.ripemd160(data);

// HASH160 (SHA256 + RIPEMD160, used in Bitcoin)
const hash160 = wallet.utils.hash160(data);

// BLAKE2b (Polkadot)
const blake2b = wallet.utils.blake2b(data, 32);

// BLAKE2s
const blake2s = wallet.utils.blake2s(data, 32);

Encoding Functions

javascript 
const bytes = new Uint8Array([1, 2, 3, 4]);

// Hex encoding
const hex = wallet.utils.encodeHex(bytes);    // "01020304"
const decoded = wallet.utils.decodeHex(hex);  // Uint8Array

// Base58 (Bitcoin-style)
const b58 = wallet.utils.encodeBase58(bytes);
const decodedB58 = wallet.utils.decodeBase58(b58);

// Base58Check (with checksum)
const b58check = wallet.utils.encodeBase58Check(bytes);
const decodedB58Check = wallet.utils.decodeBase58Check(b58check);

// Bech32 (SegWit addresses)
const bech32 = wallet.utils.encodeBech32('bc', bytes);
const { hrp, data } = wallet.utils.decodeBech32(bech32);

// Base64
const b64 = wallet.utils.encodeBase64(bytes);
const decodedB64 = wallet.utils.decodeBase64(b64);

Curve Operations

javascript 
// secp256k1 ECDSA signing
const message = wallet.utils.sha256(data);
const signature = wallet.curves.secp256k1.sign(message, privateKey);
const isValid = wallet.curves.secp256k1.verify(message, signature, publicKey);

// Recoverable signature (used in Ethereum)
const { signature: sig, recoveryId } = wallet.curves.secp256k1.signRecoverable(message, privateKey);
const recoveredPubKey = wallet.curves.secp256k1.recover(message, sig, recoveryId);

// Ed25519 signing (Solana, Polkadot)
const edSig = wallet.curves.ed25519.sign(message, privateKey);
const edValid = wallet.curves.ed25519.verify(message, edSig, publicKey);

// ECDH key exchange
const sharedSecret = wallet.curves.secp256k1.ecdh(myPrivateKey, theirPublicKey);
